# kanban.nvim

## Configuration
To change the default configuration, set `vim.g.kanban`.

Default config:
```lua
vim.g.kanban = {
  sources = {},
  highlights = {
    ListTitle = { fg = "#89B482" },
    ListBorder = { fg = "#D4BE98" },
    ListBorderFocused = { fg = "#EA6962" },
    TaskBorder = { fg = "#D4BE98" },
    TaskBorderFocused = { fg = "#EA6962" },
  },
}
```
